1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd upprompt
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   ```

   ```bash
   npm run dev
   ```

5. **Open your browser**
